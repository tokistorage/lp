What is RSS?

RSS (Really Simple Syndication) is a way to automatically receive updates from websites. By adding a feed URL to your RSS reader, you'll be notified when new articles are published. No email registration required.

How to Subscribe

Using Feedly

  1. Go to feedly.com and create an account (Google sign-in available)
  2. Click the "+" button in the left menu
  3. Paste https://tokistorage.github.io/lp/feed.xml into the search field
  4. Click "Follow"

Using NetNewsWire (Mac / iPhone)

  1. Install NetNewsWire from the App Store
  2. Go to File → New Web Feed...
  3. Enter https://tokistorage.github.io/lp/feed.xml as the URL
  4. Click "Add"

Other Readers

Most RSS readers have an "Add Feed" or "Subscribe" option where you can simply paste the URL above.

Browser Extensions

Installing an RSS browser extension lets you subscribe with a single click.

  • Chrome: "RSS Feed Reader", "Feedly Mini", etc.
  • Firefox: "Feedbro", "Livemarks", etc.
  • Safari: Works with the NetNewsWire app

Update Frequency

Essays are published almost daily, and newsletters are released monthly. The RSS feed is updated whenever new content is published.
