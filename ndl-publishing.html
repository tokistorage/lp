サービス概要

トキストレージは、国立国会図書館法（第25条・第25条の4）に基づき、 ニュースレターを逐次刊行物として国立国会図書館に納本しています。 受付番号 EA202602160140

本サービスは、この納本パイプラインをお客様（式場・イベント主催者・企業）にも開放し、 個別オリジナルのニュースレターを発行・納本するためのフルマネージドサービスです。

お客様ごとに専用の「別冊特集ニュースレター」シリーズを作成。
例: 「山田家 別冊特集」「日野ブライダル 別冊特集」——
TokiQRが届くたび、そのシリーズの1号として国会図書館に納本されます。

通常の納本申請との違い

国立国会図書館への納本は、原理上は誰でも申請できます。 では、トキストレージを通す意味は何でしょうか。

通常の自主申請 TokiStorage経由
素材の収集 手動で原稿・写真を集める TokiQRで音声・画像・テキストを自動収集
PDF制作 自分でレイアウト・PDF化 蓄積データから自動生成
NDL申請 自分でフォーム入力・提出 申請代行（手続きゼロ）
継続発行 毎号、手作業の繰り返し 献本対象を選ぶだけで即発行
公開アーカイブ 自前でホスティング GitHub Pagesで永続公開
核心はTokiQRです。紙のQRコードを配るだけで、参加者の声が自動的に集まり、 ニュースレターの素材になり、国会図書館に届く。この導線がトキストレージにしかありません。

1. セットアップ（ワンタイム）

  • クライアント専用の「別冊特集ニュースレター」シリーズ名を決定
  • 専用リポジトリを自動作成、GitHub Pagesでアーカイブ公開
  • Wise決済で完了。以降の月額費用・発行費用はゼロ

2. QR配布 → 声が届く

  • TokiQRコードを配布（結婚式、イベント、店頭など）
  • 参加者がQRをスキャンして音声・画像・テキストを記録
  • 生成時に「献本する」チェックボックスを選択するだけでパイプラインへ

3. 即発行（NDL永久保存）

  • TokiQR PDF が別冊特集ニュースレターの1号として即時発行（通巻番号・発行情報を自動付与）
  • GitHub Pagesで公開されたPDFをNDLが定期自動収集
  • 国立国会図書館法に基づく法的な永久保存
  • ISSN取得もオプションで代行可能

ユースケース

ウェディング

結婚式場・ホテルが、挙式でTokiQRコードを配布。 新郎新婦や参列者の声を記録し、献本対象に選べば逐次刊行物として国立国会図書館に永久保存。 「あなたの誓いの言葉が、国家の記録になる」——他にない体験価値を提供できます。

企業・団体

創業記念、社員の声、イベント記録をニュースレター化。 企業のCSR活動として、または社史の一部として国会図書館に残せます。

個人・家系

一族の系譜、家訓、先祖の声——代々受け継がれてきた記録を ニュースレターとして国立国会図書館に永久保存。 口伝や家伝は時間とともに失われますが、国家の記録として納本されれば、 血筋と想いは制度によって守られ続けます。

よくある質問

国立国会図書館とは？
日本唯一の国立図書館であり、国立国会図書館法に基づき 出版物の収集・保存を行う機関です。納本された資料は永久に保存されます。
発行頻度は？
TokiQR生成時に「献本する」を選ぶと、1号として即発行されます。 バッチ処理や手動の発行作業は不要です。
ISSNは必要ですか？
ISSNは任意です。NDL納本にISSNは必須ではありませんが、 学術的な位置づけを求める場合は取得をお勧めします。代行手数料は¥10,000です。
お客様側の技術的な準備は必要ですか？
一切不要です。リポジトリ作成からPDF生成、NDL納本まで すべてトキストレージが代行します。
納本されたニュースレターは一般に公開されますか？
はい。NDLに納本された資料は原則として一般に公開されます。 また、GitHub Pages上でもPDFが公開されます。
