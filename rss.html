RSSとは

RSS（Really Simple Syndication）は、ウェブサイトの更新情報を自動的に受け取るための仕組みです。RSSリーダーにフィードURLを登録すると、新しい記事が公開されたときに通知を受け取ることができます。メールアドレスの登録は不要です。

登録方法

Feedlyの場合

  1. feedly.com にアクセスし、アカウントを作成（Googleアカウントで登録可）
  2. 左メニューの「+」ボタンをクリック
  3. 検索欄に https://tokistorage.github.io/lp/feed.xml を貼り付け
  4. 「Follow」をクリック

NetNewsWireの場合（Mac / iPhone）

  1. App StoreからNetNewsWireをインストール
  2. メニューの「File」→「New Web Feed...」を選択
  3. URLに https://tokistorage.github.io/lp/feed.xml を入力
  4. 「Add」をクリック

その他のリーダー

ほとんどのRSSリーダーで「フィードを追加」または「購読」の機能から、上記のURLを入力するだけで登録できます。

ブラウザの拡張機能

ブラウザにRSS拡張機能を入れると、ワンクリックで購読できます。

  • Chrome：「RSS Feed Reader」「Feedly Mini」など
  • Firefox：「Feedbro」「Livemarks」など
  • Safari：NetNewsWireアプリと連携
